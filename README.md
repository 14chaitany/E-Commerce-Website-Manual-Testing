# E-Commerce Website Manual Testing Project

## Overview

This project demonstrates Manual Testing activities performed on an E-Commerce Web Application.

The testing process included:

* Test Scenario Preparation
* Test Case Design
* Functional Testing
* UI Testing
* Regression Testing
* Defect Reporting
* Test Execution Reporting

## Modules Tested

### Login Module

* Valid Login
* Invalid Login
* Blank Credentials Validation

### Product Module

* Product Display Validation
* Add to Cart
* Remove from Cart

### Checkout Module

* Successful Checkout
* Mandatory Field Validation

### Logout Module

* Logout Functionality Verification
